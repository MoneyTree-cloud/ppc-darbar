Implementation

The chatbot is designed for portability. A single script tag is all that's needed to bring it to life on any website.

Embed Snippet

<script src="https://your-domain.com/assets/js/embed.js"
  id="real-estate-chatbot" 
  data-website-id="your-website-id">
</script>

Runtime Attributes

  • data-theme-color: Injects a primary color (hex code).
  • data-position: Sets display to "right" or "left".
  • data-initial-message: Overrides the default greeting text.
