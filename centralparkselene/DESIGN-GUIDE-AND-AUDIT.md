### Key Discrepancies to Address

1. **Unit Count: 391 vs 353+38** -- Our page should say "353 Serviced Residences + 38 Retail Units (391 total)" rather than "~391 residences." The tilde helps but conflating retail with residential units is misleading.

2. **Size Range: 1,100-1,959 vs 1,000-3,600** -- The official press range is broader. Consider noting "select residences up to 3,600 sq ft" or qualifying our range as "standard unit" sizes.

3. **Legal Developer Entity** -- Add "Developed by St. Patricks Realty Pvt. Ltd. (Central Park Group)" in the RERA/disclaimer section.

4. **Sector 48 Confusion** -- 99acres lists Selene under "Sector 48" which is actually Central Park Resorts' registered address (corporate HQ), not the project location. Our Sector 32 is correct for the physical site.

### 1. Fix Font Cascade (Critical)

The global `* { font-family: 'Optima' }` overrides everything. Poppins and Montserrat never render.

```css
/* REPLACE THIS: */
* { ... font-family: 'Optima'; ... }

/* WITH THIS: */
* { ... font-family: 'Poppins', -apple-system, BlinkMacSystemFont, sans-serif; ... }

/* Then set headings explicitly: */
h1, h2, h3, h4, .common-heading, .projectname, .typology {
    font-family: 'Optima', 'Playfair Display', Georgia, serif;
}
```
